                                                     DESCRIPITION : COOKIE JAR
Project Description: Cookie Jar CLI Application

The Cookie Jar is a simple command-line Python application that allows users to manage a virtual jar of cookies. It demonstrates basic programming concepts such as:

Functions
Conditional statements
Loops
User input handling
String formatting

The program maintains a cookie balance and lets users:

View how many cookies are in the jar
Add cookies
Take cookies out
Exit the program

It also includes input validation to ensure users cannot:

Add negative cookies
Remove more cookies than available
Enter invalid values
⚙️ Features
✅ Track current cookie balance
➕ Add cookies to the jar
➖ Remove cookies from the jar
⚠️ Error handling for invalid inputs
🔁 Continuous menu loop until exit
🧑‍💻 How to Use
1. Run the Program

Make sure you have Python installed, then run:

python project.py
2. Menu Options

After running, you will see:

************ Cookie Jar **********

1. Number of cookies available
2. Add cookies
3. Take out cookies
4. Exit
3. Choose an Option
👉 Option 1: Check Balance
Displays current cookies in the jar

Example output:

You currently have 5 cookies in the jar
👉 Option 2: Add Cookies
Enter a positive number

Example:

Enter number of cookies to add: 3
Added 3 cookies to the jar

❌ If number ≤ 0:

Number of cookies must be greater than zero
👉 Option 3: Take Out Cookies
Enter number of cookies to remove

Example:

Enter number of cookies to take out: 2
Took out 2 cookies. Now 3 cookies left.

❌ If number > balance:

You don't have that many cookies

❌ If number ≤ 0:

Number of cookies must be greater than 0
👉 Option 4: Exit

Stops the program:

Exiting... 🍪
